# NTU_General_physics2016

## Description

These are the vpython assignments for **NTU_General_physics** class.  
This repository contains the *implementation codes* and *problem description* in each directory.

### [Assignment3](https://github.com/juichiehchang/NTU_General_Physics2016/assignment3-juichiehchang)

Simulation of a `ball-string system`.

### [Assignment4](https://github.com/juichiehchang/NTU_General_Physics2016/assignment4-juichiehchang)

Simulation of the `solar system`.

### [Assignment5](https://github.com/juichiehchang/NTU_General_Physics2016/assignment5-juichiehchang)

Simulation of `collisions`.

### [Assignment6](https://github.com/juichiehchang/NTU_General_Physics2016/assignment6-juichiehchang)

Simulation of `entropy and equlibrium`.

### [Assignment7](https://github.com/juichiehchang/NTU_General_Physics2016/assignment7-juichiehchang)

Gas, pressure and mean-free-path. Simulation of a `gas system`.

### [Assignment8](https://github.com/juichiehchang/NTU_General_Physics2016/assignment8-juichiehchang)

Simulation of the motion in a `dipole field`.

### [Assignment9](https://github.com/juichiehchang/NTU_General_Physics2016/assignment9-juichiehchang)

Simulation of the `electric field of a uniformly charged ring`.

### [Lab8](https://github.com/juichiehchang/NTU_General_Physics2016/lab8-juichiehchang)

Simulation of a `space voyage`.

### [Lab9](https://github.com/juichiehchang/NTU_General_Physics2016/lab9-juichiehchang)

Simulation of a `spring-mass system`.
